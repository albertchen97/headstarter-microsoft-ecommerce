# headstarter-microsoft-ecommerce
An e-commerce website named "Contuso" that allows users to purchase car parts. 

# Tool & Frameworks
* Tool: [AWS Amplify](https://aws.amazon.com/amplify/)
* Frameworks:
  * Frontend: React.js
  * Backend: Node.js
* Payment API: [Stripe API](https://stripe.com/docs/payments/payment-methods)
